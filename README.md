# cartao_visita

Projeto desenvolvido na matéria de Desenvolvimento para dispositivos móveis
Onde foi criado um layout em Flutter com os objetivos:

✓ Praticar criação de widgets StatelessWidget 
✓ Utilizar widgets de layout (Column, Row, Container) 
✓ Estilizar textos com TextStyle 
✓ Aplicar padding e margin para espaçamento 
✓ Compreender a árvore de widgets na prática 
✓ Versionar código com Git e publicar no GitHub

Utilizando a linguagem dart

## Requisitos necessários para iniciar

• Flutter SDK instalado e configurado 
• Editor de código (VS Code ou Android Studio) 
• Git instalado 
• Conta no GitHub

## Como Iniciar o projeto

-- Faça download do código no repositório

-- Extraia a pasta

-- Abra o aplicativo git Bash

-- Navegue até onde está a pasta baixada (provavelmente downloads) com 'cd downloads' e após isso 'cd cartao_visita'

-- Agora que esta dentro da pasta aplique o comando 'code .'

-- No VS Code aperte 'Ctrl + J' para abrir o terminal

-- Digite 'flutter pub get'

-- E por fim 'flutter run' agora pode escolher a opção 2 para rodar no EDGE
